# pwa-horn-benchmarks
Storage for benchmarks for testing my horn solver and comparing it to other horn solvers. Specifically, these benchmarks are horn clauses that model PWAs to verify convergence properties

- Kernel Version: 5.15.167.4-microsoft-standard-WSL2

- RAM: 15GB
